## 1. Цель работы
Разработка программного комплекса для верификации интерфейсов библиотеки OpenBLAS (уровень 2).

## 2. Реализованные тесты
В рамках работы были написаны интерфейсные тесты для следующих групп функций:

| Группа | Описание | Типы данных |
| :--- | :--- | :--- |
| **GEMV** | Умножение матрицы на вектор | S, D, C, Z, B |
| **SYMV / HER** | Операции с симметричными/эрмитовыми матрицами | S, D, C, Z |
| **TRMV / TRSV** | Треугольное умножение и решение систем | S, D, C, Z |
| **GER / GERC** | Ранговые обновления общего вида | S, D, C, Z |
| **SYR / HER** | Симметричные/Эрмитовы обновления Rank-1 | S, D, C, Z |
| **SYR2 / HER2** | Симметричные/Эрмитовы обновления Rank-2 | S, D, C, Z |

## 3. Архитектура тестового окружения
Проект разделен на логические модули по типам операций. Сборка осуществляется с помощью единого **Makefile**, который:
1. Рекурсивно проходит по всем подпапкам.
2. Компилирует C++ тесты с линковкой динамической библиотеки `libopenblas.so`.
3. Поддерживает массовый запуск тестов через `make run` с корректной установкой `LD_LIBRARY_PATH`.
